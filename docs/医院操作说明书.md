# 智能随访手表 · 医院操作说明书

**给医生 / 护士看的傻瓜手册**

> 这份说明书覆盖：**下载小程序 → 找 MAC 地址 → 给患者绑定 → 解绑换人**。
> 全程不需要懂技术，按步骤来即可。

---

## 你需要准备什么

| 物品 | 数量 | 说明 |
|------|------|------|
| 智能手表（S101 / VP- / VPR 系列） | 1 只 / 患者 | 已充电至少 30 分钟 |
| 手机（患者本人的） | 1 部 | iPhone 或安卓都行，能装微信即可 |
| 体验版二维码 | 1 张 | 工程师发给医院联系人，可截图、可打印 |
| 这份说明书 | 1 份 | 你正在看 |

---

## 第 1 步：下载小程序（1 分钟）

1. 让患者**用自己的微信**扫"体验版二维码"
2. 第一次扫会弹"加入体验版"提示 → 点 **确定**
3. 进入小程序首页，**右上角**会出现橙色字样：

   > **测试版 test-2026.04.28-v2**

   能看到这行字 = 小程序已正确打开。

> ⚠️ 如果扫码后提示"未授权，请联系开发者"——说明这个微信号还没加白名单，把患者的微信昵称或手机号反馈给工程师，加白名单后再扫一次。

---

## 第 2 步：找到手表的 MAC 地址（**最关键，仔细看**）

每只手表都有一个**唯一的 12 位身份编号**叫 **MAC 地址**，长这样：

```
FA:BA:94:8A:70:75
```

> 6 段两位数字字母，用冒号分隔。**前 3 段是厂商前缀**（比如 `FA:BA:94`），**后 3 段是这只表的唯一编号**。

**为什么要先找到 MAC？** —— 医院给患者发表前，需要把 MAC 登记到病历系统，这样后台才能把数据**对上是谁的**。

### 三种找法（任选一种，建议先试方法 1）

#### 方法 1：手表包装盒 / 表背贴纸（最快）

- 翻开手表**包装盒底部**或**说明书第一页**：通常印着 `MAC: XX:XX:XX:XX:XX:XX`
- 或者**翻过来看表背**：靠近表带连接处会有一行小字
- 拿手机摄像头**拉近拍照**——MAC 字号很小，肉眼可能看不清

> 📝 把照片发给医院联系人或贴在病历上做底备份。

#### 方法 2：手表自带菜单（不用连手机）

S101 的找法（不同型号略有差异）：

1. 按手表**右侧按键唤醒屏幕**
2. **从右往左滑** → 找到 **设置** 图标（齿轮形状）→ 点进去
3. 翻到 **关于** 或 **系统信息** → 里面有 **MAC** 这一行
4. 字号小，靠近看清楚再抄下来

> 不同固件版本菜单位置可能不同，**找不到就用方法 1 或方法 3**。

#### 方法 3：连上小程序后看（最稳妥）

完成下方"第 3 步：绑定"后：

- 进入首页 → **第 2 行就是 `MAC地址：XX:XX:XX:XX:XX:XX`**
- 直接抄下来 / 截图保存

> 这种方法的好处是**百分百准确**，因为 MAC 是从手表自己上报的。
> 包装盒贴纸偶尔会贴错（不同表混料），所以**最终以小程序里显示的 MAC 为准**。

## 第 3 步：给患者绑定手表（约 3 分钟）

### 3.1 打开手机蓝牙 + 网络

- iPhone：上滑控制中心 → 蓝牙图标变蓝
- 安卓：下拉通知栏 → 蓝牙开
- 同时确认手机有 **WiFi 或 4G**（数据要联网才能上传）

### 3.2 在小程序里扫描手表

1. 打开**智能随访**小程序首页
2. 点 **设备扫描** 按钮（蓝色大按钮）
3. 等 5 秒左右，列表里会出现一行：

   ```
   名称：S101            RSSI：-49
   MAC：FA:BA:94:8A:70:75
   ```

4. **核对 MAC 是否和你登记的一致** —— 如果一致，点这一行
5. 看到"**连接中…**"转圈约 3-5 秒
6. 自动跳转到**首页**，4 个字段全部填上：

   ```
   设备名称：S101
   MAC地址：FA:BA:94:8A:70:75
   固件版本：00.12.02.00-2120
   电池电量：59
   实时步数：496  431m  301kcal
   ```

✅ 看到这 4 行 = **绑定成功**。

### 3.3 iOS 第一次会弹"蓝牙配对请求"

iPhone 会弹一个系统对话框：

> **"S101" 想与你的 iPhone 配对**

**一定要点"配对"**。点取消会失败。

### 3.4 测一次血压验证

让患者：
1. 把手表戴上手腕
2. 按手表按键找到"血压"功能 → 点"开始测量"
3. 等 30 秒左右，手表显示 `高压 / 低压 / 脉搏` 三个数字
4. 在小程序里**等 10 秒**（手表数据自动同步）
5. 后台数据库可以查到这条血压记录 = 数据链路全通

---

## 第 4 步：解绑 / 换患者使用

### 4.1 解绑（不再用这只表）

1. 在小程序首页 → 点右上角 **断开连接** 按钮
2. 看到"已断开"提示 = OK
3. 关闭小程序

> 解绑后**手表本机数据保留 3 天**，3 天内重新绑定可以补传。

### 4.2 换患者（同一只表给下一个患者用）

**这只表换给新患者前，必须做的事：**

1. **当前患者**：先按上面 4.1 解绑
2. **当前患者**：iPhone 用户额外做一步 —— **设置 → 蓝牙 → 找到 S101 → 点"忽略此设备"**
   - （不做这步的话，新患者连同一只表会被旧 iPhone 抢走）
3. **新患者**：按"第 1 步~第 3 步"重新扫码、绑定
4. **后台**：把这只表的 MAC 改挂到新患者名下（联系工程师改库或用医生端界面操作）

> ⚠️ **同一只表的 MAC 永远不变**（出厂烧死的），所以"换人"不需要换 MAC，只需要把 MAC 在数据库里换个病历归属。

## 第 5 步：常见问题速查表

| 问题 | 解决办法 |
|------|----------|
| 扫描列表里看不到我的表 | 1) 充电 30 分钟；2) 表靠近手机 1 米内；3) 关再开手机蓝牙 |
| 一直显示"连接中" | 等 15 秒看会不会自动跳；超过 30 秒还在转 → 杀掉小程序重开 |
| 连上但 4 字段全空 | 重新进首页等 10 秒 / 杀小程序重开一次（不是绑定问题，是首次同步慢） |
| iOS 配对弹框点了取消 | 设置 → 蓝牙 → 找到 S101 → 忽略此设备 → 重新绑定 |
| 量了血压小程序里没数据 | 数据是给医生后台看的，**患者侧暂不显示历史**（后续版本会加） |
| 手表显示电量 0 但小程序显示 59 | 显示的是**最近一次同步时的电量**，最新值在重连后约 5 秒会刷新 |
| 给新患者用前要做什么 | 见上面"第 4 步 4.2 换患者"，**iOS 必须做"忽略此设备"** |

## 隐私说明

- 本小程序**只在佩戴期间**采集生理数据（心率、血氧、血压、体温、步数等）
- 数据通过加密通道（HTTPS）传到医院科研数据库，**不用于任何商业用途**
- 患者随时可以告知主治医生**立即解绑**

---

## 附：MAC 地址速记小贴士

> 实在记不住"MAC 地址"这个词，就记成 **手表的"身份证号"**：
>
> - **6 段两位** 字母数字，用 `:` 分开
> - 长这样：`FA:BA:94:8A:70:75`
> - **每只表唯一**，**永不变**（除非送修重刷固件）
> - **iOS 系统蓝牙**里看不到 MAC，**只能在小程序里看**或**包装盒上看**
> - 后台用它来确认"这条数据是哪只表测的、对应哪个患者"
