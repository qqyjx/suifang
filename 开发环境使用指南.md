# 智能随访项目 - BLE SDK 开发环境使用指南

## 环境说明

本项目运行在 **WSL (Windows Subsystem for Linux)** 环境中，配合 **Windows 版微信开发者工具** 使用。

## 开发环境配置

### 工具分工

| 工具 | 运行环境 | 用途 |
|------|----------|------|
| **VSCode** | WSL | 代码编写、文件管理、Git 操作 |
| **微信开发者工具** | Windows | 预览、模拟器、真机调试、蓝牙测试 |

### 工作流程

```
WSL VSCode 编辑代码 → 保存 → Windows 微信开发者工具自动热重载 → 查看效果/真机调试
```

---

## 第一步：安装微信开发者工具（Windows）

1. 访问 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 下载 **Windows 64 稳定版**
3. 安装并启动

---

## 第二步：导入项目

1. 打开微信开发者工具
2. 点击「导入项目」
3. 项目目录选择（**使用 WSL 网络路径**）：
   ```
   \\wsl$\Ubuntu\home\qq\WeChat_Mini_Program_Ble_SDK\code\demo\WeiXinSDKTSDemo
   ```
   或者：
   ```
   \\wsl.localhost\Ubuntu\home\qq\WeChat_Mini_Program_Ble_SDK\code\demo\WeiXinSDKTSDemo
   ```
   > 提示：可以在 Windows 资源管理器地址栏输入 `\\wsl$` 查看可用的 WSL 发行版名称

4. AppID 使用已配置的：`wx967e4063f0234e4d`（或你自己的 AppID）
5. 点击「导入」

---

## 第三步：配置 VSCode（WSL）

1. 在 WSL 终端中用 VSCode 打开项目：
   ```bash
   code /home/qq/WeChat_Mini_Program_Ble_SDK/code/demo/WeiXinSDKTSDemo
   ```

2. 推荐安装的 VSCode 扩展：
   - **WXML - Language Services** - WXML 语法支持
   - **TypeScript** - TS 支持（内置）
   - **ESLint** - 代码检查
   - **WSL** - WSL 远程开发支持（如果使用 Windows VSCode）

3. 代码编辑在 VSCode 中进行，保存后微信开发者工具会自动刷新

---

## 第四步：真机调试（蓝牙功能必须）

蓝牙功能**无法在模拟器中测试**，必须使用真机：

1. 在微信开发者工具中点击「预览」或「真机调试」
2. 用手机微信扫描二维码
3. 在手机上打开小程序进行测试

### 权限要求

确保手机已开启：
- 蓝牙
- 定位（部分功能需要）
- 微信蓝牙权限

## 测试蓝牙连接

### 1. 启动小程序后，进入「蓝牙连接」页面

路径：首页 → bleConnection

### 2. 点击扫描设备

设备会显示在列表中，格式类似：
- 设备名称
- MAC 地址
- 信号强度 (RSSI)

### 3. 点击设备进行连接

连接成功后会显示设备信息和支持的功能。

### 4. 测试其他功能

连接成功后可以测试：
- 读取步数
- 读取睡眠
- 心率测量
- 等等

详细 API 文档位于：

### 常用 API 示例

```typescript
import { veepooBle, veepooFeature } from '../../libs/vp_sdk/index'

// 扫描设备
veepooBle.veepooWeiXinSDKStartScanDeviceAndReceiveScanningDevice((device) => {
  console.log('发现设备:', device)
})

// 连接设备
veepooBle.veepooWeiXinSDKBleConnectionServicesCharacteristicsNotifyManager(
  { deviceId: 'XX:XX:XX:XX:XX:XX' },
  (result) => {
    if (result.success) {
      console.log('连接成功')
    }
  }
)

// 秘钥认证（必须先认证才能使用功能）
veepooFeature.veepooBlePasswordCheckManager()

// 读取步数
veepooFeature.veepooReadStepNumberManager({ day: 0 })
```

## 常见问题

### Q: 扫描不到设备？
- 确保设备已开机且在蓝牙范围内
- 确保手机蓝牙已开启
- 确保小程序有蓝牙权限

### Q: 连接失败？
- 检查设备是否已被其他手机连接
- 尝试重启设备和手机蓝牙

### Q: 功能 API 返回错误？
- 确保已完成秘钥认证
- 检查设备是否支持该功能（通过功能汇总包判断）

## 生产 vs 开发环境切换

小程序数据上传地址在 `miniprogram/services/dataStorage.ts` 中控制：

```typescript
// 当前配置（生产环境 — 数据写入六元 MySQL）
const WSL_SERVER_URL = 'https://dc.ncrc.org.cn/api2';

// 切换为本地开发（需先启动本地 Node.js 服务器）
// const WSL_SERVER_URL = 'http://127.0.0.1:3000';
```

**生产环境**：数据通过六元外网代理写入公司 MySQL，无需额外搭建。
**本地开发**：需启动本地 Node.js + MySQL，数据存入本地数据库，不影响生产数据。

```bash
# 启动本地开发服务器
echo "xyf" | sudo -S service mysql start
cd scripts && npm install && node health-data-server.js
# 访问 http://localhost:3000/api/status 验证
```
