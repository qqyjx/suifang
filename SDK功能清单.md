
# Veepoo SDK 功能清单

## 一、生命体征监测

### 1. 心率监测
- **实时心率测试**：启动/停止心率测量
- **自动心率监测**：设置自动测量时间段
- **心率历史数据**：查询过去3天的心率记录
- **应用场景**：运动监测、静息心率、心率异常预警

### 2. 血氧监测 (SpO2)
- **实时血氧测试**：手动测量血氧饱和度
- **自动血氧测试**：设置时间段自动测量（如8:00-22:00）
- **血氧历史数据**：按天查询血氧记录
- **应用场景**：呼吸系统监测、睡眠呼吸暂停筛查

### 3. 血压监测
- **实时血压测量**：收缩压/舒张压
- **动态血压监测**：JH58项目支持24小时动态血压
- **血压风险等级**：自动判断正常/偏高/高血压
- **应用场景**：高血压患者管理、心血管风险评估

### 4. 体温监测
- **实时体温测量**：腕部体温
- **自动体温监测**：周期性测量
- **体温历史数据**：体温趋势分析
- **应用场景**：发热监测、基础体温跟踪

### 5. 血糖监测
- **血糖测量**：配合血糖仪使用
- **血糖校准**：设备校准功能
- **血糖风险等级**：低血糖/正常/高血糖判断
- **应用场景**：糖尿病患者管理

### 6. 血液成分分析
- **血液成分检测**：多项血液指标
- **检测查询**：历史检测记录
- **应用场景**：综合健康评估

## 二、心电图监测 (ECG)

### 单导联ECG
- **实时ECG波形**：连续心电图数据
- **ECG数据读取**：历史ECG记录
- **应用场景**：心律不齐检测、房颤筛查

### 多导联ECG
- **多导联同步采集**：更专业的心电监测
- **ECG服务更新**：实时波形推送
- **应用场景**：专业心电分析、远程诊断

## 三、高级健康分析

### 1. HRV心率变异性分析
- **HRV数据采集**：1000+个RR间隔数据点
- **洛伦兹散点图**：心脏健康可视化
- **相似度计算**：与标准模式对比
- **星级评估**：健康等级评分
- **心脏健康指数**：0-100分综合评分
- **应用场景**：心血管健康评估、压力水平分析、自主神经功能评估

### 2. 睡眠监测
- **睡眠分期**：浅睡、深睡、REM快速眼动
- **睡眠质量评分**：综合睡眠质量
- **睡眠效率**：实际睡眠时间/在床时间
- **睡眠建议**：改善睡眠的提示
- **应用场景**：睡眠障碍筛查、睡眠质量改善

## 四、日常活动监测

### 1. 运动数据
- **步数统计**：每日步数
- **卡路里消耗**：能量消耗计算
- **活动时间**：运动时长
- **运动模式识别**：跑步、骑行、游泳等

### 2. 身体测量
- **体重**：体重数据
- **体脂率**：身体成分分析
- **BMI**：身体质量指数
- **应用场景**：体重管理、健身追踪

## 五、女性健康

### 经期管理
- **经期跟踪**：月经周期记录
- **周期预测**：下次经期预测
- **排卵期提示**：备孕辅助
- **健康提示**：经期健康建议
- **应用场景**：女性健康管理、备孕计划

## 六、设备管理功能

### 1. 表盘管理
- **表盘上传**：自定义表盘传输
- **表盘列表**：查看已安装表盘
- **表盘切换**：设置当前使用表盘
- **表盘删除**：删除不需要的表盘
- **背景图片**：自定义表盘背景

### 2. OTA固件升级
- **固件升级**：设备固件更新
- **升级进度**：实时显示升级进度
- **自动回连**：升级过程自动重连
- **错误处理**：升级失败处理

### 3. 设备配置
- **时间同步**：同步手机时间到设备
- **通知设置**：来电、短信、APP通知推送
- **ANCS推送**：iOS通知推送
- **勿扰模式**：免打扰时间段设置
- **抬腕亮屏**：手势控制设置
- **常灭屏功能**：ZT163项目特有

### 4. 4G服务
- **4G数据传输**：支持4G网络的设备
- **服务配置**：4G服务参数设置

## 七、数据同步与查询

### 1. 实时数据
- **实时推送**：设备主动推送测量数据
- **数据监听**：通过回调接收数据

### 2. 历史数据
- **按天查询**：今天、昨天、前天
- **数据类型**：
  - 日常数据（type=5）：步数、心率、血压等综合数据
  - 血氧数据（type=30）
  - ECG数据（type=51）
  - 睡眠数据：包含在日常数据中

### 3. 数据格式
```javascript
// 日常数据示例
{
  type: 5,
  content: [
    {
      bloodPressure: {systolic: 120, diastolic: 80},
      heartRate: 72,
      bloodOxygen: 98,
      temperature: 36.5,
      steps: 8500,
      calories: 320,
      timestamp: "2026-01-16 14:30:00"
    },
    // ... 更多数据点
  ],
  Progress: 100  // 数据传输进度
}
```

## 八、智能随访项目应用场景

### 1. 慢病管理
- **高血压患者**：24小时血压监测、用药提醒
- **糖尿病患者**：血糖监测、饮食运动追踪
- **心脏病患者**：心率、ECG、HRV综合监测

### 2. 术后康复
- **生命体征监控**：心率、血氧、体温
- **活动量追踪**：康复进度评估
- **睡眠质量**：恢复情况评估

### 3. 老年人健康管理
- **跌倒检测**：异常活动识别
- **生命体征异常预警**：自动报警
- **日常活动监测**：生活质量评估

### 4. 孕产妇管理
- **心率监测**：孕期心血管健康
- **睡眠监测**：孕期睡眠质量
- **活动追踪**：孕期运动管理

### 5. 远程医疗
- **数据自动上传**：实时同步到医院系统
- **医生远程查看**：随时了解患者状态
- **异常自动报警**：及时干预

## 九、技术特性

### 1. 多设备支持
- 同时连接多个设备
- 独立管理每个设备的数据

### 2. 数据可靠性
- 自动重连机制
- 数据分包传输
- MTU自适应（Android 512字节，iOS 244字节）
- 发送失败重试（最多3次）

### 3. 兼容性
- 支持Android、iOS、HarmonyOS
- 微信小程序原生开发
- TypeScript类型支持

### 4. 性能优化
- 数据缓存机制
- 异步数据处理
- 低功耗设计
